This repository contains some free downloads for K.D. Kemp's [website](https://kdkemp.com/), including book collection tracking zines and my handwriting font.

These resources are free for personal use. You don't need to credit me (you're awesome if you do), but I'd love it if you sent me an [email](mailto:hello@kdkemp.com) or messaged on [Instagram](https://www.instagram.com/k.d.kemp/) or [my guestbook](https://kdkemp.atabook.org/) to let me know if you download something!

Click **Releases** to download each book collection tracker. To print the zine, make sure you select 'scale to fit' or 'print entire image' on your printer. View printing and folding instructions at [The ZineArranger by Nash High](https://nashhigh.itch.io/zinearranger).
