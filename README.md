This repository contains some free downloads for K.D. Kemp's website, including downloadable book collection tracking zines and my handwriting font.

These resources are free for personal use. You don't need to credit me (you're awesome if you do), but I'd love it if you sent me an [email](hello@kdkemp.com) or messaged here or on [Instagram](https://www.instagram.com/k.d.kemp/) to let me know if you download something!

Click **Releases** to the right of this window to download each book collection zine. To print the zine, make sure you select 'scale to fit' or 'print entire image' on your printer. View printing and folding instructions at [The ZineArranger by Nash High](https://nashhigh.itch.io/zinearranger).
